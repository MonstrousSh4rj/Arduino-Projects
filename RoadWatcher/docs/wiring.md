# Wiring Reference

| Component       | Arduino Pin |
|-----------------|-------------|
| IR Sensor 1 OUT | Digital 2   |
| IR Sensor 2 OUT | Digital 3   |
| Buzzer (+)      | Digital 11  |
| All VCC         | 5V          |
| All GND         | GND         |

Place both IR sensors exactly 20 cm apart facing the same direction.
